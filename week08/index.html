Chapter 8: CSS3 Transforms and Transitions

Transforms:
Allow you to translate, rotate, scale, and skew html elements. Can all be done in CSS without needing JavaScript as it used to for movement. The transform property contains one or more transform function, each function separated by a space. Only declare the transform function once as having more than one assigned to an element will overwrite the first ones.

Transform functions include:
Translate functions, moves an element on the x and y axis, eg: translate(45px, -45px); To only move an element on the Y or X axis you use translateY() or translateX().
Scale functions enlarge or shrink an element using values for the Y and X axis, it still maintains the original aspect ratio of the element and has no impact on the document flow or layout. Giving a single value will apply it to both X and Y, scale(1) stays the same, scale(2) doubles the size, and scale(0.5) halves it.
Rotate function turns the element by the specified degree value, like scale the default origin is the center of the element, eg: rotate(10deg).
Skew function skews element along the X and Y axis, eg: transform: skew(15deg, 4deg);

Changing the origin of the transform: the standard origin of a transform function if the center of the element. This can be changed by using the transform-origin property

Transitions:
Animations are possible with CSS and require less client-side processing making them appear smoother, this is especially great for mobile devices because it requires less computing power than JavaScript animations. Transitions allow for animation over time, you can rotate and element as a slower speed so you can see it turn or fade a color in and out on a hover.
Transition functions are transition-property, transition-duration, transition-timing-function, and transition-delay.
Transition-property function defines the css properties that should be transitioned, things like transform, width, color, border-style.
Transition-duration function sets how long the transition will last, eg: transition-duration: 0.2s; can be written in s(seconds) or ms(milliseconds).
Transition-timing-function allows you to control the pace of a transition, you can have a transition start slow and then speed up, and any variation you want. There are common timing values used ease, linear, ease-in, ease-out, or ease-in-out. Ease is the default, starts slow speeds up and then slows down at the end, ease-in-out similar to ease but starts faster. Ease-in is slow to start then speeds up and stops abruptly, Ease-out is the opposite.
