Chapter 5: Objects

Object literals: self-contained set of related values and functions. If a property’s value is a function it is known as a method. JS objects are similar to other languages Hash or associative array or the JS Map. Are able to inherit properties similar to oop languages. Is created by wrapping all its properties and methods in {}. Key value pairs, object.keys() to get a list of the keys, object.entries() to get the values. Nested objects are possible, access them by referencing their name: object.nestedObject.key. Objects can be passed in as function parameters, allows arguments to be passed in by name and in any order.

This: refers to the object within, it can be used inside a method to get access to the to the objects properties.

Chapter 6: Document Object Model

The Document Object Model, or DOM, allows you to access elements of a webpage(html), enables interaction adding, removing, changing content, order or attributes of elements, can even alter styling.

Node list are similar to array you can find items by index number and use the length property but no other properties will work. You can turn a node list into an array using Array.from() method.

Get elements by id assigned to them in HTML: document.getElementById(‘idName’); also variations like getElementByTagName(), getElementByClassName().

Query selectors: document.querySelector() and document.querySelectorAll()
