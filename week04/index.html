Chapter 8: Forms

Form properties and methods: form.reset() sets all form controls back to initial values set in the HTML. Form.submit() submits it automatically, does not trigger the submit event.

Form events: the focus event occurs when the element is focused on, an input element if focused when the pointer is inside the element. Blur event occurs when the focus is away from the specified element. Change event happens when focus is away from the element after changing it, nothing will occur if focus is removed without change.

Input fields: text input fields include, text, password, checkbox, and radio.

Select drop-down List: used for when multiple options are needed, if you needed a user to choose one in a set of values this would be useful. Use the HTML attribute select and assign a name and id, use the option attributes tag and assign the value.
